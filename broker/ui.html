📖 Guide

Installation & mises à jour

🖥️ Serveur (le broker)

Première installation

  1. Installe Node.js ≥ 18 sur la machine serveur.
  2. Dans le dossier mailbox-handoff, double‑clique sur start-server.cmd (accepte l'élévation). Il installe les dépendances, ouvre le pare‑feu, démarre le broker et ouvre cette page.
  3. Onglet Serveur & service → Installer le service pour qu'il tourne en permanence.

Mise à jour

  1. git pull sur la machine serveur.
  2. Arrête l'ancien broker (ou désinstalle le service ici), puis re‑double‑clique start-server.cmd.
  3. Les données (messages, canaux) sont migrées automatiquement — rien à faire.

💻 Client (chaque projet / machine)

  1. Onglet Brancher un client → remplis projet + chemin (+ canaux) → Télécharger le .cmd.
  2. Place le .cmd dans le dossier mailbox-handoff de la machine cliente et double‑clique.
  3. Recharge la session Claude Code du projet → réception auto + tools MCP actifs.
Mise à jour / réparation d'un client : re‑télécharge le .cmd et double‑clique (ou .\install.ps1 -SkipProject).

📡 Canaux & 🧵 fils

  • S'abonner à un canal : ajoute-le dans « Canaux » à l'install client. Les messages du canal sont conservés côté broker.
  • Écrire à un canal : destinataire #canal — seuls les abonnés le reçoivent.
  • Répondre reste dans le même fil (bouton Répondre d'une conversation, ou tool MCP mailbox_reply).
